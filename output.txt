import os
from pyexpat import model
from xmlrpc import client
from certifi import contents
from google import genai
from google.genai import types


def analyze_text(text):
    client = genai.Client(
        api_key=os.environ.get("google_api_key"),
    )

    model = "gemini-2.5-flash-lite"

    contents = [
        types.Content(
            role="user",
            parts=[
                types.Part.from_text(text=text),
            ],
        ),
    ]

    config = types.GenerateContentConfig(
            temperature=0,
            system_instruction="""
        You are a strict JSON generator.

        Return output in this format:
        {
        "summary": "...",
        "sentiment": "positive/negative/neutral",
        "keywords": ["...", "..."]
        }

        Do not return anything outside JSON.
        """
            )

    response = client.models.generate_content(
        model=model,
        contents=contents,
        config=config
    )

    return response.text


if __name__ == "__main__":
    print(analyze_text("I hate you"))

Day 7:


Day 7 is where we stop thinking about tools and start thinking about decision-making.

Up until now your agent is:

User
 ↓
LLM
 ↓
Tool Call
 ↓
Result
 ↓
Response

This is reactive.

The model sees something and immediately acts.

But real agent systems often do:

User
 ↓
Analyze Task
 ↓
Create Plan
 ↓
Execute Step 1
 ↓
Observe
 ↓
Execute Step 2
 ↓
Observe
 ↓
Final Answer

This is the difference between a chatbot with tools and an actual agent.

Problem With Current Agent

Let's say the user asks:

Find weather in Mumbai and calculate how many days are left until New Year.

Your current agent might:

call one tool
call another tool
answer

But it has no explicit plan.

It is making decisions internally and you can't see them.

That becomes difficult when:

debugging
improving behavior
understanding failures
Goal of Day 7

We want the agent to produce:

PLAN:
1. Get current weather
2. Calculate days until New Year

ACTION:
Call get_weather

OBSERVATION:
Weather is sunny

ACTION:
Call days_until_new_year

OBSERVATION:
178 days

FINAL ANSWER:
...

This is called an agent trace.

=====Plan=====
Here's an execution plan for the task "what is 2+2?":

- **Step 1:** Identify the operation required, which is addition.
- **Step 2:** Perform the addition of the two numbers, 2 and 2, to arrive at the sum.
